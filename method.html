Data Collection Stage

For this project, the dataset that is used is a second-source data the data are collected by other organization and not myself. The dataset that is extracted can be found at oecd.org where the dataset is related to Non-medical determinants of health.

The dataset is downloaded in a comma separated value (csv) file because it is easier to process the data in that format. The file contains many data including variable, measure, country code, country name, year, value and many more.

After going though some consideration and thoughts, only a certain number of columns will be selected to be analysed. The most important data would be the alcohol consumption measures, the year it is recorded and the country it is associated with. In addition, I have derived a new column to store the continent that the country is in. This is because the countries can be grouped into continents.

As for the alcohol consumption will be recorded as a numerical data, year as an ordinal data, while country and continent as a categorical data.

Data Processing Stage

In order to filter out all the data that is not relevent and useful for the data visualisation, I have created a script of python to generate all the datasets that I need. I have used the pandas library to assist me in creating dataset

The final format of the datasets is in a comma separated value (csv) file because it would be more easier for data visualisation using D3.

From all the data from the original dataset, I have classified the data on alcohol consumption into many groups based on countries, continents and also year range. This is to ensure the dataset are classified and grouped, making it easier for data analysing to find correlations and patterns.

Data Visualisation Stage

At this stage, since all the datasets that are needed are generated, the visualisation can be made. In this project, the tool that is used to create visualisations is D3.js.

Considerations and selections are made to use the most suitable graph for each dataset. This is an essential part to ensure the data are being represented correctly and to avoid misconception.

The encoding and other factors such as hue, filtering and many more are discussed in the project book. The chart that is displayed intially are the data from 1990 and the user will be able to see data on-demand and filter the data based on continents.

The visualisations can be found at the Charts tab


Data Analysis and Conclusion Stage

At this stage, after filtering and looking the charts in many ways, findings are recorded and listed below


Findings

  • Countries in Europe has the most alcohol consumption compared to countries in other continents
  • Australia has the most alcohol consumption in Asia
  • Indonesia has the least alcohol consumption in Asia and in the world
  • African countries cannot be compared because there is only one country from Africa

Conclusion

  • Russia has the most litres of Alcohol Consumption per capita for citizens aged 15 and above since 1990 but the litres of alchol consumption has decreased over the years
  • United States is the country with the most alcohol consumption per capita for citizens aged 15 and above in the America continent since 1990
  • Lithuania who ranked 17th in Europe in 1990 for alcohol consumption per capita for citizens aged 15 and above has increased to the first rank in the 2006-2015 period
  • Australia is the only country from the Asia/Oceania continent to be ranked among other European countries in the top 50% of the list since 1990
