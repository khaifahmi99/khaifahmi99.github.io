Overview

Alcohol consumption issue has been a serious issue for many countries around the world. We kept on seeing many facts and figure that shows the problem keeps on rising and affect many people. However, do people really care about this problem?

This topic is chosen because this topic, alcohol consumption are very popular around the world. There are many warnings and awareness that has already been implemented. As this is happening, there is a question on what is the trend of alcohol consumption in some countries around the world.

The aim of this visualisation is to create figures and charts that would create awareness on the readers. As more readers are aware of this serious problem, it might have an effect hence allow this issue to be reduced

The methodology of this research are in short, gathering data of alcohol consumption from many countries and compare each countries reading with one another. Insights and findings from analysing the visualisation and aims to create a conclusion based on all the findings.

Consideration and comparison has been made in order to pick the most suitable charts that would convey the most information in a best manner to the audience. The charts and figure on alcohol consumption can be find on the Chart Tab.
